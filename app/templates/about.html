About This Project

Fantasy Football Trade Tips is my first Capstone project for Springboard's Software Engineering Track.

Challenges

Some of the main challenges in undertaking this project were:

1. The ESPN API does not have any official documentation. All endpoints used in this project were discovered either through trial and error or by reading through the codebase of another project that uses the API. This challenge helped me:

  • Solidify the importance of documentation.
  • Enhance my ability to quickly understand the purpose behind another programmers code.

2. This project required the development of an algorithm to rank and grade NFL players based on their current performance. I had to decide on a method to rank the players, and then come up with a way to scale that ranking so that players are graded based on their overall performance and the performance of other players at their position. This challenge helped me:

  • Develop an understanding of how to go about building an algorithm around real-world data
  • Enhance my ability to identify and make readily available different pieces of data in an HTTP context
